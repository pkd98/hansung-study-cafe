# 설명
## 주제
 * PHP와 부트스트랩 프레임워크를 이용하여 만든 반응형 스터디카페 홈페이지
## 기능
 1. 메인화면
 2. 회원가입, 로그인, 로그아웃, 회원정보수정
 3. 좌석 현황 보기
 4. 회원의 좌석 예약 및 퇴실기능
 5. 관리자에게 문의하기 (모달폼을 통한 쪽지보내기)
 6. 실시간 고객 소통 - PHP활용 채팅방 구현
 7. 지점위치 - 오시는길 (카카오 지도 API 사용)
 8. 관리자 페이지 - 스터디카페 현황, 문의사항 확인, 회원 강제퇴실
